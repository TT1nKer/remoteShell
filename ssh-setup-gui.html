1 Install SSH Server (Windows)

Option A: Via Settings (Windows 10 1809+ or Windows 11)

  • Press Win + I to open Settings
  • Go to Apps ‚Üí Optional Features
  • Click View Features
  • Search OpenSSH Server
  • Check and click Install
  • Wait for installation (about 1-2 minutes)

Option B: Via PowerShell (if OpenSSH Server not found)

  • Right-click Start menu ‚Üí Windows PowerShell (Admin)
  • Run this command:
Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0 Start-Service sshd Set-Service -Name sshd -StartupType 'Automatic'
üí° For Windows 7/8/8.1: OpenSSH Server is not officially supported. Consider upgrading to Windows 10/11, or use third-party SSH servers like Bitvise SSH Server or freeSSHd.
2 Start SSH Service
  • Press Win + R, type services.msc, Enter
  • Find OpenSSH SSH Server
  • Double-click to open properties
  • Change startup type to Automatic
  • Click Start button
  • Click OK
3 Configure Firewall
  • Press Win + R, type firewall.cpl, Enter
  • Click Advanced settings on the left
  • Click Inbound Rules ‚Üí New Rule
  • Select Port ‚Üí Next
  • Select TCP, Specific local ports type 22 ‚Üí Next
  • Select Allow the connection ‚Üí Next
  • Check all ‚Üí Next
  • Name: SSH ‚Üí Finish
4 Get IP Address
  • Press Win + R, type cmd, Enter
  • Type command:
ipconfig

Find IPv4 Address, write it down (e.g., 192.168.1.100)

5 Configure Router (Optional, needed for remote access)
  • Open router admin page in browser (usually 192.168.1.1 or 192.168.0.1)
  • Find Port Forwarding / Virtual Server settings
  • Add rule:
    • External port: 22
    • Internal IP: Your computer's IP (e.g., 192.168.1.100)
    • Internal port: 22
    • Protocol: TCP
  • Save settings
‚úÖ Done! Now you can connect via SSH
# Local network connection ssh username@192.168.1.100 # Remote connection (requires router configuration) ssh username@your_public_ip
‚ö†Ô∏è Note: Manual installation is only suitable for LAN use, or if you've configured router port forwarding. If you need external access without router configuration, use Tailscale.
